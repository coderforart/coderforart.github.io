相关解释：

《MWeb》第一个版本是 2015 年 1 月发布的，它的定位是 macOS 平台原生的 Markdown 编辑、笔记和静态网站生成软件。2021 年8 月发布 4.0 版本后，改名为 《MWeb Pro》。iOS 版 MWeb 好像是 2017 年发布的，比 macOS 版少很多功能，当然是没有静态博客生成功能。为了统一，2021 年，macOS 平台增加一款《MWeb 标准版》，这个版本无静态博客生成功能，跟 iOS 平台的 MWeb 一致。

《One Markdown》其实是把《MWeb》的编辑器独立出来，变成一个像 TextEdit 那样的简洁的 App。

所以最后变成：

  • 《MWeb 标准版》：支持 macOS/iPadOS/iOS 平台，Markdown 编辑、笔记、图床和发布到 Wordpress 等等一系列功能。
  • 《MWeb Pro》：仅 macOS 平台，为标准版的功能加静态网站生成功能。
  • 《One Markdown》：支持 macOS/iPadOS/iOS 平台，单纯的 Markdown 编辑功能。

要了解更多，可参考官网： https://zh.mweb.im/
