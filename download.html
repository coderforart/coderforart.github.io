版本 4.4.6

  • 修正：“生成 Github 格式的 TOC” 功能不正常的问题。

版本 4.4.5

  • 新增：文档间的引用（双链）可以在大纲视图查看了，如果文档中有引用到其它文档或被其它文档引用，则会在大纲视图中列出来。使用菜单：“视图 - 大纲视图” 可以查看大纲视图。
  • 新增：文档间的引用语法增加支持定位文档中的标题，使用办法是：在编辑器中，先把光标放到标题中，然后使用菜单：“编辑 - 复制文档引用链接” 来生成引用语法，再在其它文档粘贴即可。原本的语法为： [测试](mweblib://16292173056753)，定位到标题的语法为：[测试](mweblib://16292173056753#titile)。其中 #title 表示 “# + 文档中的标题或者文本”。
  • 新增：在快速搜索下，按 “Cmd + Shift + C” 键可以复制选中的文档的引用链接。使用说明：在写作过程中，需要引用其它文档时，可以使用 “Cmd + O” 键打开快速搜索，搜索到相关文档后，按 “Cmd + Shift + C” 键复制文档的引用链接，再按 “Cmd + V” 键就可以粘贴到当前文档了。
  • 改进：预览可以两指放大了。另外，在预览状态下，快捷键 “Cmd + +/-/0” 放大缩小字体对预览也有效果了。
  • 改进：导出 HTML 时，在使用默认主题的情况下，HTML 内容的宽度由不限制宽度改为按偏好设置中的编辑器行宽度。
  • 改进：mermaid 库升级到 10.2.3 。
  • 其它 Bug 修正及改进。

版本 4.4.4

  • 改进：对 App 的界面配色和内置主题的配色进行了改进。
  • 其它 Bug 修正及改进。

版本 4.4.3

  • 新增：专注模式。在专注模式下，界面上的所有元素都会被隐藏，只留下 "退出专注模式" 和 "置顶窗口" 两个按钮。您可以通过菜单 "视图 - 专注模式" 或工具栏上的 "眼睛图标的按钮 - 专注模式" 进入专注模式。此外，我们还在自定义工具栏面板上增加了 "专注模式" 按钮，方便您使用。
  • 改进：文档导入功能。现在，文档导入到文档库的功能已经变为独立的导入窗口。您可以选择整个文件夹进行导入，同时还可以将文件夹的结构转换为文档库的分类树。在导入前，系统会列出将要导入的文件，并以列表形式显示，方便您确认。
  • 其它 Bug 修正及改进。

MWeb 3.x 升级到 MWeb Pro 4.x

注意： MWeb 3.x 的用户都可以免费升级到 MWeb Pro 4.x，升级办法是：下载 MWeb Pro 4.x，在软件弹出的界面中会有“激活码升级”按钮，点击这个按钮即可把 3.x 的激活码兑换为 4.x 的激活码。

Mac App Store 用户： 请直接从 Mac App Store 更新即可。

提示： MWeb 2.x 升级到 4.x 需要重新付费。如果要下载 3.x 版请 点此下载 MWeb 3.x 。如果要下载 2.x 版本，请到此页面的最底部下载。
