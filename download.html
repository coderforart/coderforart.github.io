版本 4.3.8

  • 新增：现在可以把文档库中的文档置顶了，在列表中，右键要置顶的文档，选择“置顶文档”即可。
  • 新增：预览中的代码块的右上角增加了“复制”按钮。
  • 新增：快速笔记可以使用设置了的全局快捷来切换打开和关闭了，第一次按键打开，再次按键就关闭。
  • 新增：导出为图片现在可以选择导出 920 宽的图片了。
  • 修正：设置了网站发布脚本执行前的动作后却无效果的 bug。
  • 修正：当网站分类的排序为自定义拖拽排序时，生成的静态网站的“上一页、下一页”顺序不正确的 bug。
  • 更新了 mermaid 库和 mathjax 库。
  • 其它 bug 修正及改进。

版本 4.3.7 的更新

  • Bug 修正及改进。

版本 4.3.5 的更新

  • 编辑器现在可以设置多种字体了。你现在可以为标题、主要内容和代码块设置不同的字体，还可以把所设置的字体应用到 UI 和预览上。使用方法为：“偏好设置 - 通用设置 - 字体选择下拉菜单 - 新增或编辑多种字体”。
  • 工具栏上的 “眼睛” 按钮的菜单中，增加开关选项：“光标移动到链接上时显示打开按钮”。取消勾选后，光标移动到编辑器中的链接时就不会显示那个打开按钮了。
  • 其它 bug 修正及改进。

版本 4.3.3 的更新

  • 文档库修改图标的界面加上搜索图标的功能。
  • 文档库列表样式显示增加一项：“标题和内容显示成一行”，原来的“仅显示标题”样式改回原来的方式。
  • 编辑器优化打开链接的操作：当鼠标指针移动到链接上时，会在链接边上显示一个按钮，点击这个按钮就可以打开链接，不再需要按住 CMD 键再点击链接标题。
  • 修正在仅预览模式下，点击大纲视图中的标题时，一些特殊的文档会无法跳转的问题。
  • 其它 bug 修正及改进。

版本 4.3.2 的更新

  • 文档库的分类增加 “修改图标” 功能，此功能最低要求 macOS 11 。另外，在 macOS 12 上将会有 3000 多个图标可供选择。“右键分类 - 修改图标” 即可以使用此功能。
  • 优化文档库的“仅显示一行标题”的列表样式。
  • 快速笔记除了自动定期新增文档外，还增加了 “手动新增文档” 选项。选择 “手动新增文档” 后，新增文档可以点击 快速笔记 界面上的 “+” 按钮。
  • 修正了快速笔记定期新增文档时的时间点不正确的问题。
  • cmark-gfm 库更新到 0.29.0.gfm.4，这个库是用来解析 markdown 语法高亮和生成 html 的，非常重要，如果有发现任何问题，欢迎反馈。
  • 其它 bug 修正及改进。

版本 4.3.1 的更新

  • 增加：现在打开文档时，会自动滚动到上一次关闭文档时的位置。
  • 增加：从这个版本开始，非 Store 版支持自动更新了，不再需要去官网下载了。
  • 增加：LaTeX 公式增加 “数学公式自动编号” 选项，在 “偏好设置 - 通用设置 - LaTeX 的更多设置” 中 。
  • 增加：按住 option 键点击文档库的分类或外部文件夹模式的文件夹时，会 “展开/收起” 所有子分类。
  • 改进：导入主题时，如果检测到已存在的名称，会先询问是否覆盖同名的主题。
  • 现在已经不再需要区分 Big Sur 版本和 Catalina 版本了。
  • 其它 bug 修正及改进。

版本 4.2.7 的更新

  • 文档库：快速笔记可选择使用窗口方式，点击快速笔记右上角的“图钉📌”按钮即可切换为窗口方式。
  • 文档库：删除废纸篓的文档时，如果文档中的图片被其它文档所引用，则会复制一份到其它文档。
  • 其它 bug 修正及改进。

版本 4.2.6 的更新

  • 文档库的搜索除了之前的使用 -c 限定所搜索的分类外，还增加了 “右键 - 搜索此分类” 功能。搜索结果的排序有所调整，标题包含搜索关键字的会排在前面。
  • Imgur 登入改为直接打开浏览器来登入。
  • 其它 Bug 修正及改进。

版本 4.2.5 的更新

  • 偏好设置增加选项：“是否重新打开上一次退出时的窗口和标签”。
  • 文档库左下角增加 “搜索” 按钮，点击后可以打开快速搜索。
  • 文档库的文档列表样式中，仅显示标题的样式的高度调小了一些。
  • 文档库的文档列表的右键增加 “复制文档” 菜单项，使用菜单：“文件 - 复制” 也可以复制当前文档并打开。
  • 修正外部文件夹模式下，图片设置为绝对路径时，导出的 PDF 的图片不显示的问题。
  • 外部文件夹模式下，图片路径设置增加支持 {filename} 变量。
  • 修正行内数学公式使用 $ 时，包含 | 字符就不渲染问题。

版本 4.2.3 的更新

  • 文档库的 iCloud 同步按钮加上 “同步有问题？” 菜单，点击后，可以选择 “重新下载 iCloud 上文档库的内容” 或 “重新上传当前的文档库到 iCloud”。
  • 修正使用 HTML 语法中引用的本地图片在预览中不显示的 bug。
  • 快速笔记增加 “是否自动插入日期时间” 的选项。
  • 有序列表项不再会从 01. 开始了。
  • 现在可以隐藏工具条和标签条了，在视图菜单：显示/隐藏 工具条或标签条。
  • 其它 Bug 修正及改进。

版本 4.2.2 的更新

  • 修正导出为 Epub 相关问题。

  • 打开在废纸篓的文档时，编辑器上面的属性条会显示“在废纸篓中”。
  • 当勾选自动输入时自动纠正有序列表时，有序列表项数量如果在 10 以上时，将用 01. 开始。
