MWeb 2.0 测试版更新汇总

2016/7/6  posted in 版本发布  使用技巧 

目前更新情况为：7/6 2.0.1 版发布、7/8 2.0.2 版发布、7/9 2.0.3 版发布、7/11 2.0.4 版发布、7/12 2.0.5 版发布。

7/6 2.0.1 版发布

这次 2.0 版好像没什么重大 Bug，这个2.0.1 版本的更新如下：

  • 增加隐藏工具栏（视图 - 隐藏工具栏）。
  • 视图菜单增加切换编辑器主题和预览 CSS。
  • 主菜单增加发布菜单
  • 增加用快捷键切换 Tab，原来的缩进快捷键 CMD + Shift + [ 和 CMD + Shift + ] 改为前一个 Tab 和后一个 Tab，缩进快捷键请使用选中后按 Tab 和 Shift + Tab 进行前后缩进。修正选择后按 Tab 键的 Bug。
  • 增加双指左滑和右滑手势来切换三栏/仅编辑器/大纲视图。注：鼠标光标放在编辑器内才有效果。
  • 增加新建未保存的文档 CMD + Shift + N
  • 第二栏列表中增加支持 CMD + Delete 快捷键删除文档

直接下载 2.0.1：>>点此下载<<

已安装 2.0.0 版的应该会收到更新提示的，也可以手动点菜单：MWeb - 检查更新。

7/8 2.0.2 版发布

重点说明的两个改进：

2.0.2 版增加了两个要截图说明的功能，一个是表格插入和编辑的改进，请看以下 gif，可以知道大概的操作方式：

再放一张静态的图，看图即知，是全键盘操作的。

另一个就是外部模式中在编辑器右键，可以用另外编辑器打开当前的编辑的 Markdown 文档了，如下图：

另外的几个改进：

  • 发布到 Wordpress 和 MetaWeblog API 现在支持直接发布 markdown 文档了。
  • 双指左滑和右滑功能有所优化，修改了一下，另外现在不用把鼠标放在编辑器内滑动也可以了。
  • 图床功能中，之前七牛上传都会在文件名前增加日期，以防重名，现在增加了一个选项，勾选后就不会在文件名前增加日期了。
  • 之前有说一个从命令行打开的功能，然后我发现一直可以用 open -a MWeb 这个命令打开， 打开文档就是 open -a MWeb xx.md，所以这功能不用做了。

否决 PlantUML 功能

之前说的 PlantUML 功能，现在决定不做了，因为考虑到现在 MWeb 已经足够复杂了，并不合适再增加新功能，相反，要减少功能和再优化现有功能。

7/9 2.0.3 版发布

2.0.3 版主要是增加 Mavericks 支持，另外就是修正文档库拖拽排序的 bug。

直接下载 2.0.2：>>点此下载<<

7/11 2.0.4 版发布

2.0.4 版更新情况如下：

  • 修正面板切换时编辑器当前内容会跳到另外的位置的问题。
  • 修正启动后打开的文档没有恢复到上一次退出的位置。
  • 工具栏上的按钮的增加了提示。
  • 第二列的文档列表可以设置显示为两行和仅显示标题，在菜单 视图 - 列表样式 中设定。修正 通用设置 - 主题样式 中设置列表时间格式无效问题。
  • Word 粘贴到 MWeb 会变成为图片的问题，请试试使用菜单：编辑 - 粘贴为纯文本 功能。
  • 修正文档版本恢复时不能查看历史文档的 Bug。
  • 其它小修正。

直接下载 2.0.4： >>点此下载<<

7/12 2.0.5 版发布

2.0.5 版几乎是这次测试的最后一版了，这次修正有：

  • 用文档版本恢复功能时，浏览历史版本现在也可以看到图片了。
  • 右键图片，增加设置一半大小和原始大小。PS:一半大小主要是方便 Retina 的图片。
  • 表格编辑功能中，说明的文字变成可点击。比如说你点 确认 (cmd + enter) 这文字，可代替快捷键 cmd + enter。
  • 静态网站功能增加模板自带扩展（这个之后再详细说明）
  • 修正一些小 Bug。
